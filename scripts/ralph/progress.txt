## 2026-01-15 - US-002
- What was implemented: Created Running P&L Chart component with AG Charts area visualization
- Files changed: src/components/trade-detail/running-pnl-chart.tsx (new file)
- Key features implemented:
  - Fetches market data via api.marketData.getFullDayChartData
  - buildPnlTimeSeries() function that filters bars from entry to exit, calculates P&L at each bar close
  - Tracks quantity changes from scale-in/scale-out executions
  - Accumulates realized P&L from exits
  - AG Charts area series with time X-axis (HH:mm), currency Y-axis
  - Profit green / loss red fill colors using ANALYTICS_COLORS
  - Custom tooltip showing time and P&L with profit/loss coloring
  - Summary stats row: Final P&L, Peak P&L, Max Drawdown
  - Loading state with Loader2 spinner
  - Empty state when no market data available
  - Handles open trades (no exitTime) by showing up to last available bar
- **Learnings for future iterations:**
  - When iterating through an array with index checks, TypeScript requires explicit undefined checks even with length guard
  - Solution: assign array element to a variable before the while condition to satisfy type checker
  - AG Charts tooltip renderer uses inline HTML strings with style attributes for coloring
  - The buildPnlTimeSeries pattern can be reused for similar P&L visualization needs
