# Strategy Marketplace Progress Log

## Codebase Patterns
- Use Drizzle ORM for all database operations
- All constants go in src/lib/constants/ (never hardcode)
- Terminal design: monospace fonts, chartreuse accent (#d4ff00)
- Protected routes use tRPC protectedProcedure
- Cover images use existing S3 + useImageUpload pattern (see src/hooks/use-image-upload.ts)
- Rate limiting via @upstash/ratelimit with sliding window
- Auto-save pattern: useDebouncedCallback with 500ms delay
- Strategy schema in src/server/db/schema.ts (strategies table)
- Strategy router in src/server/api/routers/strategies.ts
