## Codebase Patterns
- Use Drizzle ORM for all database operations in `src/server/db/schema.ts`
- All constants go in `src/lib/constants/` (never hardcode)
- Terminal design: monospace fonts, chartreuse accent (#d4ff00)
- Protected routes use tRPC protectedProcedure
- ID generation uses `src/lib/shared/id.ts` with `ids.entityName()` pattern
- Array columns in PostgreSQL use `.array()` modifier on text columns
- Schema changes: Edit schema.ts then run `bun run db:push --force` to skip data-loss prompts
